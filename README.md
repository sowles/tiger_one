# tiger_one

## Getting Started

Execute
```
npm install openai dotenv
```

Create a .env variable with:

```
API_KEY = [Sawyers API-key no brackets]
```

### IMPORTANT
Create a .gitignore with this:
```
/node_modules/
.env
*.log
.vscode/
.idea/
.DS_Store
package-lock.json

```
Crucial that we include the .env so we dont publish our key.

## How to use
Run the file by executing:
```
node app.mjs    
```
